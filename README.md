---

## 📦 Установка

### 🔧 Через Unity Package Manager

1. Открой Unity  
2. Перейди: `Window → Package Manager`  
3. Нажми **+** → **Add package from Git URL...**  
4. Вставь следующую ссылку:

```
https://github.com/NikolaiKochergin/yandex-games-sdk.git
```
